### backend

Backend feito em node com Fastify utilizando SOLID

### como rodar o projeto
```javascript
  npm i
  cp .env.example .env
  npm run dev
  npm run test:watch
```

### techs usadas

- Typescript
- Node
- Fastify
- Prisma
- Docker
- PostgresSQL
- Eslint + Prettier
- zod
- dotenv
